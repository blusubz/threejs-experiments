Three.js Rotating Icosahedron

A simple learning project using Three.js that renders a rotating icosahedron in the browser. 

Description 

This project demonstrates the basics of Three.js:

- Creating a scene, camera and renderer
- Adding a 3D object (icosahedron)
- Applying material and color 
- Rendering to the screen
- Animation rotation (Would need to comment out line in code to turn on add this feature)
- Adding controls 

Getting Started 

1. Clone the repository

git clone <your-repo-url>
cd <your-folder>

2. Install dependencies 

npm install

3. Run the project

npm run dev


What I learned 

- How the Three.js render pipeline works (scene -> camera -> renderer)
- How to create and display 3D geometry 
- Basic animation using requestAnimationFrame
- Camera positioning and object visibility 

Tech Used 

- Three.js
- JavaScript (ES6)
