# Earth (Three.js)

A 3D Earth built with Three.js featuring realistic textures, clouds, atmosphere glow and a dynamic starfield.

# Features
- Textured Earth (diffuse, bump, sepcular map)
- City lights on the dark side
- Transparent cloud layer
- Fresnel-based atmospheric glow (shader)
- Rotating starfield background

# Tech 
- Three.js
- JavaScript (ES Modules)
- WebGL

# Structure 
index.html - entry point 
index.js - scene setup and animation
textures/ Earth maps and assets
src/ - helpe modules (stars, shaders)
